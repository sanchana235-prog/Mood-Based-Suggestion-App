# Project Requirement Document (PRD)

## Project Title
Mood-Based Suggestion App

## Objective
To build an AI-powered application that provides helpful suggestions and motivational support based on the user’s current mood.

## Problem Statement
Students and individuals often feel stressed, tired, or unmotivated but don’t know what small actions can help them feel better. Existing solutions are either too complex or not personalized.

## Solution
A simple web app where users input their mood, and the system generates:
- 3 actionable suggestions
- 1 short motivational/supportive message

## Target Users
- Students
- Young professionals
- Anyone seeking quick emotional support

## Features

### Core Features
- Mood input (text-based)
- AI-generated suggestions
- Motivational message display

### Optional Features
- Predefined mood buttons (e.g., stressed, tired, bored)
- Tone selection (calm, motivating)
- “Quick 5-minute action” suggestion

## User Flow
1. User enters mood
2. Clicks “Get Suggestions”
3. System processes input using AI
4. Displays suggestions and message

## Success Criteria
- App generates relevant suggestions
- Response time < 3 seconds
- Clean and usable UI
- Working demo

## Constraints
- Must be completed within a day
- Minimal UI complexity
- No authentication or database required
